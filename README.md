# Мини-проект: Документы + Чат

Проект реализован на **React + Next.js + TypeScript** с использованием Tailwind CSS.  
Состоит из двух страниц: 

1. **Документы**
   - Таблица с колонками: Название файла, Версия, Размер, Дата загрузки, Действие.
   - Данные загружаются из публичного API (`jsonplaceholder.typicode.com/posts`).
   - Кнопка **“Подробнее”** открывает модальное окно с информацией о документе.
   - Кнопка **“Проанализировать”** имитирует POST-запрос и выводит сообщение "Анализ выполнен".

2. **Чат**
   - Простая реализация с мок WebSocket (echo-сообщения).
   - Сообщения пользователя справа, сообщения бота слева.
   - Скролл вниз при новых сообщениях.
   - Поле ввода текста и кнопка отправки сообщений.

---

## Запуск проекта

1. Клонировать репозиторий:

```bash
git clone (https://github.com/omodotori/test-mini-docs-chat.git)
```

2. Перейти в папку проекта:
```
cd my-doc-chat
```

3. Установить зависимости:
```
npm install
```

4. Запустить проект:
```
npm run dev
```

5. Открыть в браузере:
```
http://localhost:3000
```
